Welcome to the Payment Processing System API

This is a simple static page served by the Node.js backend.

The core functionality is exposed via RESTful API endpoints. You can interact with the API using tools like Postman, Insomnia, or by building a dedicated frontend application.

API Endpoints (Examples):

  • POST /v1/auth/register - Register a new user
  • POST /v1/auth/login - Log in and get JWT tokens
  • GET /v1/users/<userId> - Get user details (requires authentication)
  • POST /v1/accounts - Create a new bank account (requires authentication)
  • GET /v1/accounts - Get user's bank accounts (requires authentication)
  • POST /v1/transactions - Initiate a payment transaction (requires authentication)
  • GET /v1/transactions - Get user's transaction history (requires authentication)

Refer to the API Documentation for detailed endpoint specifications.

For development, access the API on http://localhost:3000/v1 (or your configured port).
