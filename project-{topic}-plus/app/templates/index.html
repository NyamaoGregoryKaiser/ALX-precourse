Welcome to the Machine Learning Utilities System

This system provides a robust platform for managing your machine learning lifecycle, from dataset registration to experiment tracking and model deployment.

Key Features:

  • Dataset Management: Register, describe, and track your training and validation datasets.
  • Model Management: Version and deploy your machine learning models with detailed metadata.
  • Experiment Tracking: Log hyperparameters, metrics, and artifacts for every experiment run, ensuring reproducibility and easy comparison.
  • User Authentication & Authorization: Secure access with JWT and role-based controls.
  • Scalable Architecture: Built with FastAPI, PostgreSQL, and Docker for production readiness.
