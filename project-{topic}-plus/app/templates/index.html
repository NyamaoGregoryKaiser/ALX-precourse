Welcome to the ALX Enterprise CMS!

This is a comprehensive Content Management System built with Flask, SQLAlchemy, and a suite of modern web technologies.

It includes features like:

  • User Management with Authentication (JWT) and Authorization (Role-based)
  • Content Management for Posts and Categories
  • Media Asset Management
  • Database Migrations (Alembic)
  • Caching (Redis)
  • Rate Limiting
  • Structured Logging
  • Comprehensive Error Handling
  • Containerization with Docker
  • CI/CD Pipeline example
  • Extensive Testing (Unit, Integration, API)

Getting Started:

Please refer to the README.md file in the project root for detailed setup and usage instructions.

You can interact with the API endpoints directly using tools like Postman or curl, or build a separate frontend application to consume these APIs.

API Endpoints (Examples):

  • POST /api/auth/register - Register a new user
  • POST /api/auth/login - Authenticate and get JWT tokens
  • GET /api/users - Get all users (Admin only)
  • GET /api/posts - Get all posts
  • POST /api/posts - Create a new post (Authenticated, Editor/Admin)
  • GET /api/categories - Get all categories
  • ... and many more!

Explore the project structure to understand the backend logic, database models, and API definitions.
