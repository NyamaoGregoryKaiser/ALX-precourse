From here, an ADMIN can:

**Important for testing:** The `admin/dashboard` and `admin/users` routes are secured with `ROLE_ADMIN`. If you navigate directly and are not authenticated or not an ADMIN, you'll be redirected to login or get an access denied error. Please log in as 'admin' (username: admin, password: password) to view these pages.
