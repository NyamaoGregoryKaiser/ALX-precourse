Welcome to the ML Utilities System

This is a backend application designed to manage Machine Learning datasets, features, and models.

API Documentation (Swagger UI)

Explore and interact with the API endpoints:

  • Go to Swagger UI

Note: API endpoints require authentication (JWT Bearer Token). Use the `/api/auth/signin` endpoint to obtain a token first.

Authentication Endpoints

  • POST /api/auth/signup: Register a new user
  • POST /api/auth/signin: Authenticate and get a JWT token

Data Management Endpoints (CRUD)

All data management endpoints require authentication.

  • Datasets: /api/datasets
  • Features: /api/features
  • Models: /api/models

Detailed CRUD operations and required roles are documented in Swagger UI.

Project Information

  • Backend Framework: Spring Boot 3.x (Java 17)
  • Database: PostgreSQL (Flyway for migrations)
  • Authentication: JWT with Spring Security
  • Caching: Spring Cache with Caffeine
  • Rate Limiting: Custom Interceptor with Bucket4j
  • Testing: JUnit 5, Mockito, Spring Boot Test, Testcontainers
  • Documentation: Springdoc OpenAPI (Swagger UI)
  • Deployment: Docker, Docker Compose
  • CI/CD: GitHub Actions

For detailed setup and deployment instructions, please refer to the project's README.md file on GitHub.
